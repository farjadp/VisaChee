# Changelog

All notable changes to the "VisaChee" project will be documented in this file.

## [1.0.0] - 2024-02-06

### Added
-   **Core Bot Architecture**: Implemented `python-telegram-bot` with `ConversationHandler`.
-   **Path A (Quick Scan)**: 5 rapid-fire questions with immediate "Match/No Match" feedback.
-   **Path B (Deep Dive)**: 4-Level gamified assessment covering Profile, Treasure, Strategy, and Final Boss.
-   **Scoring Engine (`logic.py`)**: Weighted scoring system for NL, FI, DK, UAE, and CA.
-   **Content Module (`questions.py`)**: Centralized text management for easy copy editing.
-   **Restart Functionality**: Ability to restart the assessment at any time.
-   **Admin Contact**: Link to contact admin in final results.
